# Bank-Credit-Debit-Analysis
📊 Bank Debit & Credit Analysis Dashboard – Tableau

This repository contains an interactive Bank Debit & Credit Analysis Dashboard built using Tableau Desktop. The dashboard visualizes key transaction trends across multiple bank branches and helps understand the flow of debit and credit activities over time.

🏦 Project Overview

The dashboard provides a comprehensive view of banking transactions by breaking down credit, debit, and net transactions across branches, time periods, and transaction methods.
It aims to help users quickly analyze:

How branches are performing

Month-wise transaction patterns

Distribution of payment methods

Credit-to-Debit relationship

Overall trends and imbalances

This project demonstrates data visualization, dashboard design, and analytical storytelling.

✨ Key Features
1. KPI Summary Cards

Net Transaction

Credit to Debit Ratio

Total Debit Amount

Total Credit Amount

These metrics offer an instant high-level performance snapshot.

2. Filters for Better Interactivity

Users can analyze data by:

Transaction Type (Credit / Debit)

Branch

Month of Transaction

Bank Name
Includes a custom Reset All Filters button for convenience.

3. Visualizations Included
📌 Total Transaction by Branch

Horizontal stacked bars showing both debit and credit amounts for each branch.

📌 Transactions Over Time

Line chart illustrating monthly debit and credit trends, capturing seasonality and anomalies.

📌 Net Transaction by Branch

Positive and negative bars indicating which branches lead or lag in net transactions.

📌 Transaction Method Analysis

Pie chart displaying the share of:

Credit Card

Debit Card

Bank Transfer

🎯 Insights Highlighted

Branch-wise contribution to total banking activity

Month-on-month changes in transaction behaviour

Identification of highly active and low-performing branches

Customer preference for different payment modes

🛠️ Tools Used

Tableau Desktop Public Edition – Dashboard creation & visualization
